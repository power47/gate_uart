Тестовое задание: шлюзование данных между двумя интерфейсами UART.

Исходные коды проекта расположены в директории src.
API для работы с периферийными модулями UART размещено в файле uart.h.

Алгоритм буферизации данных размещён в файле buffer.c.


Unit-тесты алгоритма буферизации данных размещены в директории tests.
Сборка проекта с тестами произаодится компилятором GCC по команде make.


Указанные API и алгоритм собраны в проект Keil uVision 5.34.
Файл проекта размещён в директории uart.
Аппаратная платформа проекта - плата STM32F4-Discovery.
